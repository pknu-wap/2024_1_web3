ㅁi니왑ㅍi
===========
>WAP의 소셜 네트워크 서비스

🤷소개
------------
ㅁi니왑ㅍi는 WAP + 미니홈피로 WAP회원들이 각자 개인 홈피를 소유하여
회원들끼리 소통하고 글을 게시할 수 있으며, WAP의 소통을 마이크로 블로그 서비스를 제공합니다.
